# Face Mask Detection using Convolutional Neural Networks (CNN)

## Overview

This project uses a Convolutional Neural Network (CNN) to classify whether a person is wearing a face mask or not from facial images. The model is trained on labeled image data and learns visual features automatically through convolution and pooling layers.

The objective of this project is to explore computer vision and deep learning techniques for real-world safety and compliance applications.

## Features

* Face Mask vs No Mask classification
* Image preprocessing and normalization
* CNN-based feature extraction
* Model training and evaluation
* Performance visualization using accuracy and loss metrics

## Dataset Used 
[https://www.kaggle.com/datasets/omkargurav/face-mask-dataset]

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## Project Workflow

1. Load and extract the image dataset
2. Preprocessing the image data
3. Build CNN architecture
4. Train the model on mask and no-mask images
5. Evaluate model performance
6. Generate predictions on unseen images
